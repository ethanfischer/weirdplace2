This project uses [DlgSystem](https://github.com/NotYetGames/DlgSystem.git).

In order for the project to open correctly without errors, you must clone the `UpgradeTo5.1` branch into the Plugins folder:

```
cd Plugins
git clone -b UpgradeTo5.1 https://github.com/NotYetGames/DlgSystem.git
```

Then, when you open the project in unreal, it should prompt you to build DlgSystem before the project opens. Do this and you should be good
